# TechNEX_zuluu - Advanced Mutual Fund AI System

AI-powered mutual fund recommendation system with modern UI components and enhanced data visualization.

## Features

- **AI Recommendations** with 96.4% accuracy
- **789 Mutual Funds** from 39 AMCs analyzed
- **Real-time Predictions** using machine learning
- **Modern UI Components** with shadcn/ui integration
- **Enhanced Data Visualization** with interactive pie charts
- **Light/Dark Theme Support** with seamless switching
- **Professional Dashboard** with performance stratification
- **Complete Analysis** with correlations and insights

## Architecture

- **Backend**: FastAPI with ML models (Python 3.12+ compatible)
- **Frontend**: Next.js 16 with TypeScript and shadcn/ui
- **UI Components**: Modern shadcn/ui component library
- **Styling**: Tailwind CSS 4 with theme system
- **Charts**: Enhanced Recharts with custom components
- **Database**: CSV data with 789 funds
- **ML Models**: Gradient Boosting & Extra Trees

## Quick Start

### Backend (Python 3.12+ Compatible)
```bash
cd backend

# Automatic installation (recommended)
python install.py

# Or manual installation
pip install -r requirements.txt
python run.py
```

### Frontend (Modern UI Stack)
```bash
cd frontend
npm install
npm run dev
```

### Access Points
- **Frontend**: http://localhost:3000
- **Backend API**: http://localhost:8000
- **API Documentation**: http://localhost:8000/docs
